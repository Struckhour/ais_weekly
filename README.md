This is the R analysis code and data for the paper "The phenology of environmental DNA Detection for coastal Aquatic Invasive Species."

Data is imported and prepared in `AIS_eDNA_data_prep.R`.

Most of the scripts run this file at the top, but if you find you're missing data, a good check is to make sure this file has been run.

Code for the analysis and production of figures and tables is in `/code_2026/`. It is organized into subfolders that are similar to sections of the manuscript.

- `Detection windows/` for all analyses of seasonal timing of eDNA concentration

- `Environmental properties/` for figures and analyses of how environmental properties interact with eDNA concentration

- `plate_and_life_stages/` for analyses of how abundance on plates and life stages interacted with eDNA concentration

- `map/` contains code for generating the map in the paper

- `Exploration Code/` contains code that was not directly used for analyses and figures in the manuscript, but was used for exploring the data and inspiring some analyses.
